# Django 프로젝트 시작 방법

```bash
1.$ pyenv local intro-venv
 #가상 파이썬
1.1 django 설치
2.$ django-admin startproject first_workshop .  (프로젝트작성)
3.$ python manage.py startapp pages  (앱 실행)
4.templates 폴더 생성
5.settings 들어가서 allowed_hosts, installed apps 설정변경(~~.apps.~~Config)
6.$ python manage.py runserver $IP:$PORT
```
